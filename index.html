💡 คำแนะนำสำหรับ GitHub Pages

เนื่องจาก GitHub Pages เป็น Static Site:

  • ข้อมูลจะเก็บใน localStorage ของแต่ละเบราว์เซอร์
  • ใช้ปุ่ม "ส่งออกข้อมูล" เพื่อบันทึกเป็นไฟล์ JSON
  • ใช้ปุ่ม "นำเข้าข้อมูล" เพื่อโหลดไฟล์ JSON
  • แชร์ไฟล์ JSON กับคนอื่นเพื่อความเป็นเอกภาพ
  • สำหรับเว็บที่ใช้ร่วมกัน แนะนำให้ใช้ Firebase/Supabase
